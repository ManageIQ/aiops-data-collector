## Configure

* `AWS_ACCESS_KEY_ID` - AWS S3 credentials
* `AWS_SECRET_ACCESS_KEY` - AWS S3 credentials
* `AWS_S3_BUCKET_NAME` - AWS S3 bucket selector
* `NEXT_MICROSERVICE_HOST` - where to pass the collected data (`hostname:port`)
